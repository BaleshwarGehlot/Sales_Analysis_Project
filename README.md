# Sales_Analysis_Project

This project performs a detailed sales analysis using Python. 
It includes data cleaning, exploratory data analysis (EDA), visualization of trends, 
and insights on top products, categories, regions, and customers. 
The analysis also evaluates shipping efficiency and highlights delayed orders.

## Dataset

- The dataset `sales.csv` contains sales records with columns such as:
  - Order ID, Customer Name, Product Name, Category, Sub-Category
  - Sales, Order Date, Ship Date, Shipping Mode, Region, City, State
- It is used to perform sales trend analysis, customer analysis, and shipping evaluation.

## Analysis Performed

- Data cleaning and handling missing values
- Conversion of date columns and calculation of shipping duration
- Sales analysis by Category, Sub-Category, Region, City, and Product
- Top customers analysis (by sales and order count)
- Shipping performance and delayed order detection
- Visualizations:
  - Pie charts for sales distribution (Category, Sub-Category, Region, Top Products)
  - Bar charts for Top Cities and Top Customers
  - Line chart for Monthly Sales Trend
  - Histogram for Shipping Duration
  - Heatmap for Monthly Sales by Category
